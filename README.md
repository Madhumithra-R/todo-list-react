# ✅ Todo List App with Local Storage

A simple and functional Todo List application built with **React.js**.

🔗 **Live Demo:** https://madhumithra-r.github.io/todo-list-react/

---

## 🛠️ Tech Stack

- React.js
- Vite
- CSS (No external UI libraries)

---

## ✨ Features

- Add a task
- Mark task as completed (strikethrough)
- Delete a task
- Filter tasks — All / Active / Done
- Show total task count
- Persist tasks using localStorage

---

## 🚀 Getting Started
```bash
# Clone the repo
git clone https://github.com/Madhumithra-R/todo-list-react.git

# Go into the project
cd todo-list-react

# Install dependencies
npm install

# Run locally
npm run dev
```

---

## 📦 Deploy
```bash
npm run deploy
```
